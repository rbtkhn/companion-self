# Start here (companion-self template)

Plain-language entry for **this template repository**. A **live instance** with a full Record and bots is separate (e.g. **[Grace-Mar](https://github.com/rbtkhn/grace-mar)**). Precise terms: [concept.md](concept.md), [identity-fork-protocol.md](identity-fork-protocol.md).

## In one sentence

Companion-self is the **portable constitution** for a governed companion **Record** — seed formation, gated updates, and change-review discipline — not a copy of someone else’s fork.

### Coming from OB1-style systems?

Use **[Coming from OB1? Start here](start-here-ob1-users.md)** for a translation table (Library, Skills, Evidence, Approval Inbox) and links back to the reference instance.
