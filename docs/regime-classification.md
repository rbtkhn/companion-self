# Regime Classification

This document defines the transition classes used by the Lawful Succession layer.

The purpose of classification is to distinguish small governed updates from deeper transitions that affect continuity.

## Classification set

### Administrative update

A low-impact change that does not materially alter durable governed state.

Typical traits:
- local scope
- low contradiction burden
- easy rollback
- no constitutional effect

Examples:
- clarification of wording
- small preference refinement
- formatting or naming cleanup
- low-risk profile precision improvements

### Reform

A meaningful but continuous change.

A reform updates the companion in a way that remains recognizably continuous with prior doctrine.

Typical traits:
- moderate scope
- meaningful evidence basis
- continuity preserved
- reviewable rationale
- no break in governing identity

Examples:
- clearer pedagogy preference with evidence
- refinement of long-standing curiosity priorities
- stronger articulation of an already-behavioral trait

### Amendment

A structural durable change to governed rules.

An amendment affects not only what the companion contains, but how it is governed.

Typical traits:
- high scope
- strong evidence requirement
- preserved prior state required
- explicit decision required
- affects future merge logic or doctrine

Examples:
- memory-governance rule change
- new threshold for post-seed review
- revised boundary doctrine
- structural pedagogy doctrine change

### Fork

A legitimate branch rather than a single replacement.

A fork is appropriate when two future continuities are both intelligible and one should not erase the other.

Typical traits:
- dual legitimacy possible
- continuity preserved on both branches
- divergence is explicit rather than hidden
- branch identity is reviewable

Examples:
- public-facing branch and private branch
- teaching branch and strategy branch
- family branch and work branch

### Restoration

A lawful return to an earlier doctrine, mode, or rule.

A restoration is not mere rollback. It is a justified re-adoption of a prior state with preserved context.

Typical traits:
- references prior state explicitly
- explains why restoration is appropriate
- preserves intervening history
- avoids pretending the intermediate state never happened

### Coup-risk

A proposed change that attempts to exercise high authority with weak legitimacy.

Typical traits:
- high impact
- weak evidence
- low continuity
- insufficient preserved prior state
- process bypass or overwrite tendency

Examples:
- silent replacement of a core identity commitment
- major merge without preserved rationale
- automation-driven structural change with weak support

### Collapse-risk

A condition where continuity becomes difficult to govern coherently.

Typical traits:
- unresolved contradiction density is high
- multiple structural rules conflict
- active doctrine is unclear
- succession path is no longer intelligible

Examples:
- multiple competing identity regimes
- incompatible governance rules operating at once
- accumulated changes that no longer compose into a readable whole

## Why classification matters

Classification determines:
- required review tier
- depth of preserved evidence
- continuity burden
- whether branching is preferable to replacement
- whether the change can be treated as lawful

## Non-goals

This document does not yet assign numeric thresholds.

It defines the conceptual categories only.
