# Glossary — companion-self (short)

**Full instance glossary example:** [Grace-Mar glossary](https://github.com/rbtkhn/grace-mar/blob/main/docs/glossary.md). **Protocol compact:** [identity-fork-protocol.md](identity-fork-protocol.md).

| Term | Definition |
|------|------------|
| **Approval Inbox** (user-facing) | Plain-language name for **pending candidates** in `users/<id>/recursion-gate.md` before merge — same queue as **recursion-gate** ([approval-inbox-spec.md](approval-inbox-spec.md)). |
| **recursion-gate** (gate) | Staging surface: candidates until companion (or delegated human) approves; then merge tooling applies. |
| **OB1** (informal) | Shorthand for **Open Brain**–style personal knowledge / gateway mental models; onboarding bridge only. |
| **companion-self** (hyphenated) | The **template** repository — constitution, scaffolds, validators — not a live person’s Record ([README.md](../README.md)). |
| **Instance** | A repo (e.g. [Grace-Mar](https://github.com/rbtkhn/grace-mar)) created from the template with a real `users/<id>/` after seed phase. |

See also: [start-here-ob1-users.md](start-here-ob1-users.md).
